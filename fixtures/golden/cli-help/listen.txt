Usage:
  claudio listen [options]

Options:
  --tempo <DOUBLE>            declared tempo in BPM (auto-estimated if omitted)
  --out-dir <PATH>            directory to write raw.mid/score.mid/score.musicxml (default: out)
  --view                      open a live sheet-music browser view
  --record                    also write input.wav + recreation.wav
  --note-names                print a scientific-pitch-name lyric under each note
  --time-signature <STRING>   time signature for the score (e.g. 3/4, 6/8; default 4/4) (default: 4/4)
  --soundfont <PATH>          explicit SoundFont for the --record recreation (auto-discovered otherwise)
  --mono                      use the monophonic YIN pipeline instead of the polyphonic default
  --separate                  route through source separation before transcribing each stem (prototype-quality; see DECISIONS.md)
  --include-vocals            with --separate, fold the vocal stem into the piano score/recreation (always a multitrack.mid track regardless)

Example:
  claudio listen --view --record
